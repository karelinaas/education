Абстрактные классы в Python — это как "шаблон" или "чертёж" для других классов. Они нужны, чтобы "заставить" дочерние классы реализовывать обязательные методы.

Таким образом код будет работать как задумано (все наследники точно будут иметь нужные методы), будет более чёткое разделение общей логики и конкретной реализации, ну и заодно другие разработчики будут видеть, какие методы должны быть реализованы)))

Важный момент: нельзя создать экземпляр абстрактного класса!

В нашем случае это могло бы выглядеть так:

```python
from abc import ABC, abstractmethod

class BaseCat(ABC):
    @abstractmethod
    def meow(self) -> str:  # Обязательный метод для всех наследников
        raise NotImplementedError

    @abstractmethod
    def purr(self) -> str:
        raise NotImplementedError

    # Какие-то методы мы можем не делать абстрактными, а реализовать прямо тут.
    # Например, в тех случаях, когда мы точно знаем, что во всех наследниках
    # метод будет делать в точности одно и то же.
    def sleep(self):
        return "хр-р-р"

    
class ExoticShortHair(BaseCat):
    def meow(self):
        return "МААААУУУУ"
    
    def purr(self):
        return "мррррррррррррррррр"
```

@classmethod имеет возможность работать с классом, не задавая объекта, компилятор питона сам добавляет cls (ссылка на класс), позволяя работать с переменными класса (допустим мы определили переменную класса test_var) и с ней мы можем при помощи этого метода напрямую взаимодействовать.

@staticmethod это вспомогательная функция, которая не может взаимодействовать на прямую с данными объекта (self не может), а вот если передать, то какие-то данные можно получить

```python
import typing


# Задание 9. Магические методы
# Реализуй в этом классе такие магические методы, чтобы сниппет кода в конце
# работал без ошибок и делал то, что написано в комментариях.

class BaseCat:
    name: str
    age: int
    weight: typing.Optional[float]
    length: int

    BREED = 'кот домашний'

    def __init__(
        self,
        name: str,
        age: int,
        weight: typing.Optional[float] = None,
    ):
        self.name = name
        self.age = age
        self.weight = weight

    def __len__(self) -> int:
        return self.length

    def __call__(self):
        print("Ты чего делаешь? Я котик, а не функция!")

    def __add__(self, other: int):
        self.weight = self.length + other
        return self

    def __str__(self):
        return f"Это кот домашний по имени {self.name}"

    def meow(self) -> str:
        return 'Мяу!'

    def purr(self) -> str:
        return 'Мррррррр...'


cat = BaseCat('Уголёк', 2, 4.1)
cat.length = 70
print(cat) # должно вывести 'Это кот домашний по имени Уголёк.'
print(len(cat))  # должно вывести '70'
cat()  # должно вывести 'Ты чего делаешь? Я котик, а не функция!'
cat = cat + 2
print(cat.weight)  # должно вывести '72'
```
